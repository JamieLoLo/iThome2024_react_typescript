### 流程

1. 下載專案

```
git clone https://github.com/JamieLoLo/iThome2024_react_typescript.git
```

2. 安裝相關套件

```
npm install
```

3. 啟動伺服器

```
npm run dev
```

4. 開啟瀏覽器，並前往以下：

```
http://localhost:5173/
```
